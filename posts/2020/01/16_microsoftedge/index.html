从 pkg 安装包中提取 MicrosoftEdge APP

Jan 16, 2020 • 预计阅读时间 1 分钟

微软的新 Edge 终于发布了，全面使用 Chromium 内核，对于 Web 开发来说是好事，以后不用单独适配和测试。

官方的下载地址：

https://www.microsoft.com/en-us/edge

Windows 和 macOS 都已经是正式版本

macOS 的安装包是 pkg 格式的：

下载安装包

Google 的 Chrome 提供的安装包是 dmg 格式，直接解压拖进 /Applications 目录就能用了。

微软的这个 New Edge 安装程序居然是 pkg，在安装过程中应该是执行脚本搞些什么事情。

但对于我们来说，只需要里面的 APP，其它额外操作都不是必要的，而且人家 Google 都不搞 pkg，微软小动作太多了。

在终端下执行命令解包 pkg：

pkgutil --expand-full MicrosoftEdge-79.0.309.65.pkg MicrosoftEdge-79.0.309.65

进入 MicrosoftEdge-79.0.309.65 目录里，看到：

解包后的目录结构

然后「显示包内容」：

里面的目录结构：

Scripts 是存放脚本的目录，感兴趣可以看看，但对我们没有什么影响。

Microsoft Edge.app 才是我们需要的，直接拖进「应用程序」目录里就可以了。

启动台里的图标

点击运行，主界面这样的：

New Edge

看一下关于信息：

关于页面

扩展商店，只能从微软提供的商店里选择扩展：

扩展页面

Enjoy it.
