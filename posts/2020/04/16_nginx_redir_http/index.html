Nginx 重定向 HTTP 请求到 HTTPS 端口

Apr 16, 2020 • 预计阅读时间 1 分钟

如果配置了 nginx 只允许 HTTPS 访问，在浏览器里不加 https:// 访问的话就会报错，添加一个 301 重定向 https 可以解决问题。

server {
        listen ${HTTP_PORT} ssl http2;
        ssl_certificate     /opt/certs/server.crt;
        ssl_certificate_key /opt/certs/server.key;

        error_page 497 301 https://$http_host$request_uri;
}
