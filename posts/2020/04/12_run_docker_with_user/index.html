Ubuntu 以普通用户启动 Docker

Apr 12, 2020 • 预计阅读时间 1 分钟

以下操作在 Ubuntu 18.04 中执行。

创建一个普通用户

root 身份下添加一个普通用户 lvv：

useradd lvv

设置密码：

passwd lvv

设置默认 Shell：

usermod -s /bin/bash lvv

让用户可以执行 sudo，方便安装软件和系统维护：

usermod -G sudo -a lvv

以后就使用 lvv 这个用户来登录系统。

安装 Docker

Ubuntu 下 Docker 的包名是 docker.io：

sudo apt-get install docker.io

安装完后会自动创建一个同名的用户组，然后把用户 lvv 添加到 docker 组里就可以了：

sudo usermod -G docker -a lvv

修改以后该用户重新登录权限才会生效，以后再操作 docker 的时候就不需要先使用 sudo 提权了，对于安全运维很有用。
