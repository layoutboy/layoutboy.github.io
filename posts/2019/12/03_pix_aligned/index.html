视网膜屏幕下像素对齐计算方法

Dec 03, 2019

使用 CoreGraphics 裁剪图片的时候如果计算后的尺寸没有像素对齐，得出的图像就会有 1px 的黑边。

例如需要计算出 0.5 的高度在当前屏幕下需要多少像素进行显示，像素对齐的算法如下：

let scale = UIScreen.main.scale
let h = ceil(0.5 * scale) / scale

使用 UIKit 的方法 UIGraphicsBeginImageContextWithOptions 则会对传入的 size 进行像素对齐。

使用代码进行验证：

UIGraphicsBeginImageContextWithOptions(CGSize(width: 0.5, height: 0.5), false, 0)
let img = UIGraphicsGetImageFromCurrentImageContext()
UIGraphicsEndImageContext()

输出 img.size 的值：

iPhone 8 ：0.5 x 0.5
iPhone 8 plus ：0.6666666 x 0.6666666
