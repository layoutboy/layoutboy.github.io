Fix Reveal load error in iOS 12 (jailbreaked)

May 02, 2019 • 预计阅读时间 1 分钟

After install CCRevealLoader, RevealServer.framework installed at /Library/Application Support/CCRevealLoader/RevealServer.framework

file system sandbox blocked mmap When loading Reveal server

a solution: move RevealServer.framework to /Library/TweakInject/

mv "/Library/Application Support/CCRevealLoader/RevealServer.framework" /Library/TweakInject/
ln -s /Library/TweakInject/RevealServer.framework /Library/Application\ Support/CCRevealLoader/

That`s OK now.
