Git 操作技巧

Nov 20, 2019 • 预计阅读时间 1 分钟

  • 删除远程仓库所有 tag

    git push origin --delete $(git tag -l)

    执行后将删除远程仓库的所有 tag。
    不过，本地已经拉取回来的 tag 还在。

  • 创建一个没有任何提交记录的分支

    git checkout --orphan branche-name
