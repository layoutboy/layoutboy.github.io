Nov 07, 2019

码云

官网： https://gitee.com

功能最为丰富，Pages 需要付费才能使用自定义域名和 https

Coding

官网： https://coding.net

已被腾讯收购，Pages 功能免费提供自定义域名和 https

阿里云

官网： https://code.aliyun.com

简洁易用，目前还没有 Pages 功能

官网： https://git.weixin.qq.com
